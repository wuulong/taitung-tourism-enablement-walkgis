# 卑南遺址公園 Peinan Site Park (Peinan archaeological park)

這是一個位於台東市的核心點位，代表了地方生活的真實質感與深厚的文化脈絡。

## 📜 歷史背景
此點位於台東市區域，與卑南溪流域的發展息息相關。透過長期的在地經營，已經成為當地社區與旅客之間重要的情感連結點。
